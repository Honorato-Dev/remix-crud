## Getting started

* Dependency

  - Express JS
  - bcrypt
  - json web token
  - sql
  - react-dom
  - zod
 
  
  
* To install the dependencies.
```bash
  $ npm install
  ```
  
* To run the project.
```bash
  $ npm run start
  ```
